# Protótipo - Sistema de Cadastros

Projeto desenvolvido como protótipo para atividade acadêmica.

## O que faz
- CRUD básico (Adicionar / Editar / Excluir)
- Busca por nome/email
- Persistência local via `localStorage`
- Estrutura simples baseada no padrão MVC + Repository (para protótipo)

## Tecnologias
- HTML / CSS / JavaScript (Vanilla)

## Como executar
1. Clone:
   `git clone https://github.com/miranha11111121312/porjeto-sistema-cadastro.git`
2. Abra `index.html` no navegador (ou use Live Server no VS Code).

## Observações técnicas
- **Linguagem interpretada:** JavaScript é interpretado pelo navegador.
- **Linguagem compilada:** Ex.: Java, C# (compiladas para bytecode/binário).
- **Frameworks x Ferramentas:** frameworks (React, Angular) ajudam a estruturar apps; ferramentas (VS Code, Git, Live Server) ajudam no desenvolvimento/deploy.
- **Padrões aplicados:** MVC (separação de responsabilidades), Repository (acesso a dados encapsulado).
- **Integração com BD real:** para produção, criar API (Node/Express ou outro) que comunique com DB (Postgres, MySQL ou MongoDB). O `CadastroRepo` pode ser substituído por chamadas `fetch()` para endpoints REST.
